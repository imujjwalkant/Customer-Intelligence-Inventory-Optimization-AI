# Customer-Intelligence-Inventory-Optimization-AI
Customer Intelligence and Inventory Optimisation Using Predictive and Agentic AI

Ujjwal Kant - Liverpool John Moores University, 2026

This repository contains all source code for the research paper.

Distillbert_Experiment_1.ipynb through _8.ipynb - DistilBERT hyperparameter experiments

DistilBert_Experiment_Final.ipynb - Final large-scale DistilBERT training (Experiment 9, n=883,636)

DQN_AGENT.ipynb - Deep Q-Network inventory optimisation agent

sales_db_random.ipynb - Generate synthetic sales data.
